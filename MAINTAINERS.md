# Maverick Maintainers

Status: active public maintainer list.

## Maintainer

- GitHub: [@ilhaformosa](https://github.com/ilhaformosa)
- Role: project maintainer and release decision owner

The maintainer reviews changes, preserves the security and privacy boundaries,
triages private reports, and records release decisions. `CODEOWNERS` routes
reviews to this public identity. It does not create a second reviewer or an
independent security audit.

## Contact Boundaries

Use GitHub issues for non-sensitive project discussion and GitHub private
vulnerability reporting for security reports. Do not publish personal email,
private infrastructure, credentials, raw logs, or private evidence to contact a
maintainer.

Commit email privacy and signature rules are in
`docs/MAINTAINER_IDENTITY_AND_SIGNING.md`.
