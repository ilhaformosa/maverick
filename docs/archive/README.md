# Historical Archive

Everything below `docs/archive/` is retained for provenance only. It is not an
active plan, current status source, release gate, authorization, audit result,
or production-readiness claim.

The archive contains former:

- Phase and release plans;
- evidence, readiness, and review records;
- protocol/governance proposals;
- root-level compatibility, migration, test, and release documents;
- machine-readable blocker and production ledgers.

Do not maintain cross-document consistency inside this archive. When historical
text conflicts with `STATUS.md`, `STATUS.md` wins. Moving a file here does not
delete or relabel its historical result.
