# Pre-Publication Release Records

Every release record in this directory predates the sanitized public Git
history. The named versions were created and verified in the private
repository, but their commits, tags, GitHub Releases, Actions runs, and other
repository objects were intentionally not imported into the public repository.

URLs, commit hashes, run identifiers, and artifact hashes in these files are
historical evidence identifiers. A URL using the current public repository
name might not resolve and must not be treated as a current download or release
entry point.

Old tags must not be recreated on different public commits. See
`docs/PUBLIC_HISTORY_BOUNDARY.md` for the active rule.
