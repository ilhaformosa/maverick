# Archived Tooling

The Python coordination/validation tools and former remote/evidence shell
runners under this directory are frozen historical tooling.

They are not part of the active local gate, do not define progress, and must not
be used to restart Phase 3 or claim current product evidence. They remain in
version control so prior records can still be interpreted.

The active human-readable product entry point is `scripts/user-smoke.sh`; the
active code-quality gate is `scripts/local-harness.sh`.
